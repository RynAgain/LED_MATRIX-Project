API Documentation

REST API endpoints for the LED Matrix web control panel. All endpoints except /api/health require authentication.

Status & Health

GET
/api/health
Health check -- no authentication required. Returns status, timestamp, and version.
GET
/api/status
Display status and system stats (CPU, RAM, disk, temp, uptime).
GET
/api/system-stats
System resource usage (CPU %, RAM %, disk %, temperature, uptime).
GET
/api/preview
Live matrix preview as base64-encoded PNG image.
WS
/ws/status
WebSocket real-time status feed. Pushes JSON status every 2 seconds.

Feature Control

POST
/api/play_feature
Switch to a specific feature immediately.
Body: {"feature": "name"}
POST
/api/restart
Restart the display service via systemctl.
POST
/api/update
Trigger a git update check via the updater service.
POST
/api/brightness
Set display brightness.
Body: {"brightness": 10-100}
POST
/api/reorder-features
Reorder the feature display sequence.
Body: {"order": ["feature1", "feature2", ...]}

Configuration

POST
/api/countdown
Set and start the countdown timer.
Body: {"label": "Timer", "seconds": 300}
POST
/api/qr
Set QR code content and display it.
Body: {"content": "https://...", "label": "Scan Me"}
POST
/api/github-stats
Save GitHub stats configuration (username, token).
Body: {"username": "...", "token": "..."}
GET
/api/config-backup
Download all config files as a .zip archive.
POST
/api/config-restore
Upload and restore config files from a .zip backup.
Multipart form: field "backup" (.zip file)

Images & Pixel Art

GET
/api/images
List all slideshow images with file names and sizes.
POST
/api/upload-image
Upload an image for the slideshow. Auto-resized to 64x64.
Multipart form: field "image" (png/jpg/gif/bmp, max 5MB)
POST
/api/delete-image
Delete an image from the slideshow directory.
Body: {"filename": "image.png"}
GET
/api/image/<filename>
Serve a specific image file for preview.
POST
/api/pixel-art
Send pixel art data directly to the matrix display.
Body: {"pixels": [[r,g,b], ...]}
POST
/api/save-pixel-art
Save pixel art as a PNG image in the slideshow directory.
Body: {"pixels": [[r,g,b], ...]}

Logs & Errors

POST
/api/clear-logs
Clear the display log file.
POST
/api/dismiss-error
Dismiss the current feature error alert.

Version Control

GET
/api/git-log
Recent commit history (last 20 commits). Returns commit hashes, messages, and current HEAD.
POST
/api/git-rollback
Rollback to a specific git commit. Hash must be alphanumeric, minimum 7 characters.
Body: {"hash": "abc1234"}
